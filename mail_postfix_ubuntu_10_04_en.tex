\subsection{Postfix - Ubuntu 10.04}

Ubuntu 10.04 comes with \service*{postfix} version 2.7.0 in the main repository.

\subsubsection{Properties}

The values for the default properties can be found
in \reftbl{tbl:mail_postfix_ubuntu_10_04_properties}.

\paragraph{install\_command}

\property{mail!install\_command}{install\_command \typestring}

The command to install package on the Ubuntu system.

\paragraph{restart\_command}

\property{mail!restart\_command}{restart\_command \typestring}

The restart command for the postfix server.

\paragraph{postalias\_command}

\property{mail!postalias\_command}{postalias\_command \typestring}

The \command{mail!postalias}{postalias} command for the postfix server.

\paragraph{maincf\_file}

\property{mail!maincf\_file}{maincf\_file \typestring}

The name of the main configuration file of \service*{postfix}, usually known
as \code{main.cf}.

\paragraph{mastercf\_file}

\property{mail!mastercf\_file}{mastercf\_file \typestring}

The name of the master process configuration file of \service*{postfix}
service, usually known as \code{master.cf}.

\paragraph{configuration\_directory}

\property{mail!configuration\_directory}{configuration\_directory \typestring}

The path of the \service*{postfix} service configuration directory, usually
\code{/etc/postfix}.

\paragraph{packages}

\property{mail!packages}{packages \typelist}

The packages needed for the \service*{postfix} service.

\paragraph{bind\_addresses}

\property{mail!bind\_addresses}{bind\_addresses \literalall$|$\literalloopback$|$\typestring}

The default network interface addresses that this mail system receives mail on.
Specify \literalall{} to receive mail on all network interfaces,
and \literalloopback{} to receive mail on loopback network interfaces only.
