\subsection{Postfix - MySQL - Ubuntu 10.04}

Ubuntu 10.04 comes with \service*{postfix-mysql} version 2.7.0 in the main repository.

\subsubsection{Properties}

In addition to the properties defined in \refsec{sec:postfix_ubuntu_10_04_properties}
the \service*{postfix-mysql} service defines additional properties.
The values for the default properties can be found
in \reftbl{tbl:mail_postfix_mysql_ubuntu_10_04_properties}.

\paragraph{packages}

\property{mail!packages}{packages \typelist}

The packages needed for the \service*{postfix-mysql} service.
