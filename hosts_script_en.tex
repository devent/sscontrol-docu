\subsection{Hosts Script}

\paragraph{hosts}

\statement{hosts}{hosts \{...\}}

Entry point in the hosts script.

\paragraph{ip}

\statement{hosts!ip}{ip \typestring}

Adds a new host entry with the specified IP address.

\paragraph{host}

\statement{hosts!host}{host \typestring}

Adds a new host entry with the specified IP address.

\paragraph{alias}

\statement{hosts!alias}{alias \typelist}

Optionally, adds the specified aliases for the host.
