\subsection{Hosts Script}

\input{hosts_script_statements_en}

\TheStatement{hosts}
\TheStatement*[hosts]{hosts \{ ip host alias \}}

Entry point in the hosts script.

\TheStatement[hosts:ip]{ip}
\TheStatement*[hosts!ip]{ip \Arg{address}}

Adds a new host entry with the specified IP \Arg{address}.

\TheStatement[hosts:host]{host}
\TheStatement*[hosts!host]{host \Arg{host}}

Adds a new \Arg{host} entry with the specified IP address.

\TheStatement[hosts:alias]{alias}
\TheStatement*[hosts!alias]{alias \Arg{aliases}}

Optionally, adds the specified \Arg{aliases} for the host.
