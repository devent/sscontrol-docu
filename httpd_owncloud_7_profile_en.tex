\label{sec:owncloud_7_profile}
\subsection{ownCloud 7 Profile}

The current versions of ownCloud 7 is 
7.0.5\footnote{\printdate{2015-03-11}} from the vendor web site
\url{https://owncloud.org}.

%% owncloud_php_command
\TheProperty[httpd:owncloud_7:owncloud_php_command]{owncloud\_\\php\_command}
\TheProperty*[httpd!owncloud\_7!owncloud\_php\_command]{owncloud\_php\_command \Arg{path}}

The \Arg{path} of the \code{php} command.
Can be a full path or just the command name that can be found in the current search path. 
For the distribution
\begin{inparaitem}
\item[\TheDistribution{ubuntu}] Ubuntu 12.04
\end{inparaitem}
the value is set to \qcode{/usr/bin/php-cgi}.
