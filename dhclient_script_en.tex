\subsection{Dhclient Script}
\input{dhclient_script_statements_en}

\TheStatement{dhclient}
\TheStatement*[dhclient]{dhclient \{ requests prepend \}}

Entry point in the dhclient script.

%% requests
\TheStatement{requests}
\TheStatement*[dhclient!requests]{requests \Arg{requests}}

Sets the list of DHCP \Arg{requests}. Request item that starts with an 
exclamation mark \qcode{!} is removed from the list.

\begin{lstlisting}[style=Java]
dhclient {
    requests "!domain-name-servers"
}
\end{lstlisting}

\TheStatement[prepend]{prepend}
\TheStatement*[dhclient!prepend]{prepend \Arg{option} \Arg{declaration}}

Adds the \Arg{option} with the \Arg{declaration} as a prepend statement 
to the DHCP configuration.

\begin{lstlisting}[style=Java]
dhclient {
    prepend "domain_name_servers", "127.0.0.1"
}
\end{lstlisting}
