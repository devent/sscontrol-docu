\subsection{Hosts Script}

\input{hosts_script_statements_en}

\TheStatement{hosts}
\TheStatement*[hosts]{hosts \{ ip \}}

Entry point in the hosts script.

\TheStatement[hosts:ip]{ip}
\TheStatement*[hosts!ip]{ip \Arg{address}, host: \Arg{host} [, alias: \Arg{aliases}]}

Adds a new host entry with the specified IP \Arg{address}, the \Arg{host} 
name and optionally the host \Arg{aliases}.

\begin{lstlisting}[style=Java]
hosts {
    ip "192.168.0.52", host: "srv1.ubuntutest.xx"
    ip "192.168.0.49", host: "srv1.ubuntutest.com", alias: "srv1"
}
\end{lstlisting}
