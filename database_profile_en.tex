\subsection{Database - Profile}

%% service
\TheProperty[database:service]{service}
\TheProperty*[database!service]{service \Arg{name}}

The \Arg{name} of the database service. Can be one of the following 
service types.
\begin{asparaitem}
%
\item[\qcode{mysql}:] 
the MySQL database comes in the distributions 
\begin{compactitem}
\item[\TheDistribution{ubuntu}] Ubuntu 12.04\footnote{\TheUbuntuPreciseLTSDate} in version 5.5 in the main repository;
\item[\TheDistribution{ubuntu}] Ubuntu 14.04\footnote{\TheUbuntuTrustyLTSDate} in version 5.5 in the main repository.
Though MySQL 5.6 is available in \TheUbuntuTrustyLTSDate, the \code{mysql-server} metapackage will install the older version 5.5.
\end{compactitem}
%
\end{asparaitem}
