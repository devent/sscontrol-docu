\label{sec:roundcube_profile}
\subsection{Roundcube Profile}

The current version of Roundcube is 
1.1.0\footnote{\printdate{2015-02-08}} from the vendor web site
\url{http://roundcube.net/}.

%% roundcube_config_file
\TheProperty[httpd:roundcube:roundcube_config_file]{roundcube\_\\config\_file}
\TheProperty*[httpd!roundcube!roundcube\_config\_file]{roundcube\_config\_file \Arg{path}}

The \Arg{path} of the service configuration file. If the path is not absolute
then it is assumed to be under the service installation directory.
For the distribution
\begin{inparaitem}
\item[\TheDistribution{ubuntu}] Ubuntu 12.04
\end{inparaitem}
the value is set to \qcode{config/config.inc.php}.
